## 7. 整数计费

统一 token 口径：

```text
input_tokens = 包含 cached tokens 的总输入
cached_input_tokens = cached 子集
uncached_input = max(input_tokens - cached_input_tokens, 0)
```

公式：

```text
numerator =
    uncached_input * input_rate
  + cached_input_tokens * cached_rate
  + output_tokens * output_rate

base_cost_microusd = ceil(numerator / 1_000_000)
charged_cost_microusd = ceil(numerator * multiplier_millis / 1_000_000_000)
```

要求：

- 中间计算用 `i128`，落库前检查范围。
- 只在倍率应用后做一次最终向上取整。
- snapshot 保存实际 tier/rates、tokens、usage source、multiplier、base 和 charged cost。
- actual usage 优先；缺 usage 或流式中断时用确定性本地估算并标记 estimated，不能默认为 0。
- 价格读取、snapshot、幂等 wallet ledger 和余额更新在一个 Core transaction。
- 幂等键使用 request ID/request log ID。
- 历史 snapshot 不随改价变化。
- 旧 `estimated_cost_usd` 如需报表兼容，只从 snapshot 派生，不能参与扣费。

重点现有入口：

- `crates/service/src/gateway/observability/request_log.rs`
- `crates/service/src/auth/app_manager.rs::wallet_charge_for_request`
- `crates/service/src/quota/model_pricing.rs`
- `crates/core/src/storage/account_manager.rs`

## 14. Definition of Done

只有以下全部满足才能宣称完成：

- fresh DB 自动 seed 最新 8 个 builtin，普通模型页立即显示 7 个。
- DB/UI 只有 builtin/custom，无 remote runtime 语义。
- 所有上游 `/models` 同步链从启动、管理页、aggregate 和 Gateway 消失。
- model、price tiers、routes、groups、instructions policy 原子保存。
- Gateway 只读 V2 routes，账号/aggregate 切换无 body 污染。
- 客户端非空提示词原样传递，seed 不含官方人格提示词。
- input/cached/output、long tier、multiplier、snapshot 和 wallet ledger 全部整数可复算。
- 截图中的 remote 按钮和旧提示全部消失。
- 完整 Rust/frontend/desktop 测试通过。
- fresh DB 与 legacy DB 都完成 `cargo tauri dev` 实际验收。
- 工作区干净，最终报告列出全部 commit hash 和是否 push。

最终报告必须分别说明 migration/seed、UI、Gateway、instructions、billing、测试、Tauri dev 实际状态和提交记录。不得把“只完成后端”或“只完成 migration”表述为整个 cutover 完成。
